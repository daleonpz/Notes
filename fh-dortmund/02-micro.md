# Introduction
- NMOS vs CMOS: 
    - NMOS: static power ( vcc --[ R ]-- uC ) :> I2xR 
    - COMS: connects the vcc directly to the uC pin ( vcc --- uC )

- ECL vs CMOS: ECL is more expensive and does not have any true advantage to CMOS.
